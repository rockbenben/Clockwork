开机助手 使用说明
==================

一个 Windows 托盘小工具，管好开机与日常四件事：
  1. 我的启动清单 —— 开机自动按顺序打开常用软件、顺手做杂事
  2. 定时提醒     —— 到点提醒 / 朗读 / 重复催促 / 点「是」后做事
  3. 系统启动项   —— 查看并管理电脑里所有开机自启项，一键关闭不需要的
  4. 动作组       —— 把一串动作打包成「组」（专注 / 收工 / 睡前…），一键触发

【怎么开始】
  1. 解压到任意文件夹（绿色免安装，放哪都行）。
  2. 双击「开机助手-双击运行.bat」打开设置窗口。
  3. 想开机自动运行：右下角托盘图标右键 →「设为开机自启」
     （用管理员权限注册计划任务，开机时不会弹一堆授权框）。
  平时它待在右下角托盘；关闭窗口只是收起，不会退出。彻底退出用托盘右键「退出」。
  勾选窗口底部的「启动时最小化到托盘」后，手动打开也直接进托盘、不弹主窗口。

【第一次用：把示例换成你自己的】
  首次打开，清单/提醒/动作组里已有一批「示例」（都带「示例」字样），只是演示玩法、
  给你照葫芦画瓢用的，放心大胆改或删。启动清单里前几条默认勾选、开机会执行；末尾的
  「Windows 设置/任务管理器」等默认没勾选，只当样例摆着，勾上才执行。
  最常见的「开机自动打开我常用的几个软件」，这样弄：
    1) 切到「我的启动清单」。
    2) 把用不上的示例整条删掉（选中→删除，删完自动选中下一条，连着删很顺手）。
    3)「新增 ▾」→ 启动程序，「目标」填你要开的软件：
       - 系统能找到的填程序名即可：msedge.exe（Edge）、notepad.exe（记事本）；
       - 找不到的填完整路径：右键该程序→「打开文件所在位置」→右键图标「属性」，
         把里面的「目标」路径复制粘进来；
       - 网址（https://…）、文档、.ps1 脚本、快捷方式（.lnk）也都能填。
    4) 想某个软件晚点再开，就把它的「执行后延时」调大几秒，或上下移调顺序。
    5) 点底部「重新运行启动清单」先试跑一遍，确认都对。
    6) 满意后托盘右键「设为开机自启」，以后开机自动照做。
  只想用「定时提醒」或「动作组」？去对应标签页照同样思路改示例即可，启动清单可全删空；
  三块功能各自独立，用哪块开哪块。

【我的启动清单】
  - 一条有序步骤表，开机从上到下依次执行；可增删、上移/下移；双击条目即编辑。
  - 每步可单独启停、设「执行后延时」、设条件「仅某些星期 / 仅当天 N 点前」。
  - 选中一条按「运行」= 立即只跑这一步看效果（无视该项启停与时间条件，纯测试）；结果弹托盘气泡反馈。
  - 步骤类型：
      启动程序：exe / 文档 / 快捷方式 / 网址都行（目标可「浏览…」选文件）；.ps1 自动用 PowerShell 运行。
                工作目录留空 = 用目标所在目录。进阶：窗口风格（最小化/最大化/隐藏启动）、
                已在运行则激活窗口（不重复启动，进程名可「选择…」）、备用路径（每行一条，
                主路径不存在时用第一个存在的，方便多设备安装路径不一致）。
      发送按键：如 Win+D、Alt+K、Ctrl+Enter、F5（支持 Enter/Tab/Esc/Del/方向键等键名；可点「捕获」按快捷键自动填）
      发送文本：往当前焦点窗口逐字输入一段文本（换行=回车、Tab 生效）；可选「目标进程」——
                填了先把它的窗口带到最前再输入，留空则发给当前焦点窗口
      音量：静音 / 取消静音 / 设为音量（设音量会先自动取消静音）
      窗口动作：按进程名 关闭 / 最小化 / 最大化 / 带到最前面 / 置前并发送按键；
                慢启动应用可设「等待窗口出现 N 秒」，窗口一出现才动手、不盲等固定延时
      系统命令：显示桌面 / 锁屏（回来需输密码）/ 息屏（只关屏幕，动鼠标即亮）/
                清空回收站 / 清空剪贴板 / 打开 Windows 设置 / 打开任务管理器 /
                截图 / 睡眠（秒醒）/ 休眠（断电保存）/ 注销 / 重启 / 关机（后三项需确认）
      延时：单纯等待 N 秒再继续下一步；放清单最前面可整体推迟启动
      动作组：引用一个已定义好的动作组
  - 开机延迟：清单页底部「开机延迟 N 秒」框，仅开机自启时生效。登录后先固定等这么多
    秒、让「登录风暴」（磁盘/CPU 被一堆自启抢占）过峰，再跑清单；手动「重新运行」不受
    影响。开机太早、有软件没起来就把它调大（0–600 秒）。这是整体延时的唯一旋钮；只想
    让某一步慢一点则用那步的「执行后延时」。
    进阶：想「等网络/桌面就绪再跑」而非固定等待，可在 开机助手-设置.json 把
    startupWaitForReady 设为 true（默认 false，就绪即走、封顶 90 秒）。

【定时提醒】
  - 触发：定时（到点）或 登录时（可设「仅开机 N 分钟内」才算登录——新建默认 10 分钟）。
  - 周期：按星期 / 每 N 天 / 每月某号；提醒可朗读。
  - 没配动作的提醒 → 走系统通知（右下角滑出，错过自动进通知中心）；重复催促型仍用弹窗，好一键停催；系统通知被关掉时自动退回弹窗。
    系统通知显示时长用「自动关闭」秒数控制：≥20 秒=长通知(~25s)，否则短(~5-7s)——Windows 只这两档，做不到任意秒数。
  - 配了「点是后」动作（运行程序 / 打开文件 / 开网页 / 跑动作组）→ 弹置顶「是 / 否」窗，
    还有「稍后…」可打个盹；弹窗在后台线程，不会卡住主窗口。
  - 进阶：自动关闭（N 秒没点自动关）、重复催促（没确认就每 N 分钟再弹、直到某时刻）、
    到点后延迟 + 随机延迟、宽限（到点时恰好关机 / 睡眠，N 分钟内补弹）。
  - 「静默动作组」：到点不弹窗，直接跑指定动作组。选中提醒按「运行」立即预览。

【系统启动项】
  - 列出电脑里所有开机自启项（注册表 Run 键 / 启动文件夹 / 计划任务）。
  - 取消「启用」勾选即关闭 —— 只是禁用、不删除，随时勾回来恢复（勾选即时生效）。
  - 带「需管理员」的项：操作时会提示「以管理员身份重新打开」，同意后再操作即可。
  - 置灰的「只读」项为系统 / 策略项（组策略 Run、RunOnce、Winlogon、Active Setup），
    无法开关，仅供查看。
  - 「纳入启动清单」：把某项交给本工具接管（自动停用原项 + 加进你的清单）。
  - 项目多时，顶部「过滤」框可按名称 / 命令快速筛。

【动作组】
  - 「新增 ▾」可从内置模板（专注 / 会议 / 收工 / 睡前 / 离开一下 / 截图标注）创建，
    改成自己的软件再保存。
  - 同一组同时只会跑一份，重复触发自动跳过。
  - 触发方式：托盘右键「运行：某组」/ 启动清单加「动作组」步骤 / 提醒挂「点是后 / 静默动作组」。
  - 组里的「消息」步骤可当确认闸门：答「否」中止整组（比如收工前问"记录好了吗？"）。

【小提示】
  - 双击 bat 只打开设置窗口，不会立刻执行启动清单；要立刻跑用「重新运行启动清单」。
  - 个人设置存在文件夹内的「开机助手-设置.json」，只在你自己电脑上；
    删掉它重开程序即可恢复成示例配置。
  - 填路径/进程/组合键/日期不用硬记：目标·工作目录可「浏览…」，进程可「选择…」（带搜索），
    组合键可「捕获」，起算日可「选日期」；进程选择框和系统启动项列表都带搜索/过滤。
  - 请用正常方式启动本程序（双击 bat / 托盘 / 计划任务）。若用某些沙箱/受限权限的启动器
    （如 Lucy）打开，Windows 会限制运行时编译，发送按键/窗口动作/已运行则激活/发送文本到
    指定进程/音量 等原生功能会不可用（会给清楚提示，不影响单纯「启动程序」）。
