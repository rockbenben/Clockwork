> 365 Open-Source Plan #020 · أداة في شريط نظام Windows: مُشغّل بدء التشغيل · تذكيرات · عناصر بدء تشغيل النظام · مجموعات إجراءات

![Clockwork](../assets/social-card.png)

أداة صغيرة في شريط نظام Windows تتولّى الأجزاء الروتينية من بداية يومك أمام الحاسوب:

- 🚀 **قائمة بدء التشغيل** — افتح تطبيقاتك اليومية تلقائيًا عند تسجيل الدخول، بالترتيب (صلاحيات مسؤول لكل خطوة، تأخيرات، في أيام أسبوع محددة فقط / قبل الساعة N فقط، نمط النافذة، التنشيط إن كان يعمل، مسارات بديلة)، وقم ببعض المهام الجانبية على الطريق (إغلاق أو إبراز نوافذ، إرسال ضغطات مفاتيح / نص، ضبط مستوى الصوت…).
- ⏰ **التذكيرات** — أظهر تذكيرًا في وقته؛ اقرأه بصوت مسموع؛ كرّره حسب أيام الأسبوع / كل N يوم / شهريًا؛ أو فعّله «عند تسجيل الدخول». النقر على **نعم** يمكن أن يشغّل برنامجًا، أو يفتح ملفًا (مثل الموسيقى) أو رابطًا، أو يشغّل مجموعة إجراءات.
- 🧹 **عناصر بدء تشغيل النظام** — اعرض **كل ما يبدأ تلقائيًا على حاسوبك** وأوقف ما لا تحتاجه (مُعطّل، غير محذوف — أعده في أي وقت). نقرة واحدة «تستولي» على عنصر لتضيفه إلى قائمة بدء التشغيل الخاصة بك.
- 🎛️ **مجموعات الإجراءات** — اجمع سلسلة من الإجراءات في مجموعة قابلة لإعادة الاستخدام (تركيز / اجتماع / إنهاء اليوم / قبل النوم…) وشغّلها بنقرة واحدة من الشريط أو **مفتاح عام** أو قائمة بدء التشغيل أو تذكير. تتضمن قوالب جاهزة.

بلا تثبيت، مجلد واحد محمول بالكامل، كل شيء قابل للضبط بالفأرة؛ واجهة داكنة، متوافقة مع DPI العالي.

> 📖 **الدليل الكامل:** [English](USAGE.md) · [中文](USAGE.zh-CN.md)

## المتطلبات

- Windows 10 / 11 (x64)
- لا شيء لتثبيته: ملف واحد قائم بذاته `Clockwork.exe` مع تضمين وقت تشغيل .NET بداخله.

## البدء

1. نزّل أحدث `Clockwork-<الإصدار>.zip` من [Releases](https://github.com/rockbenben/Clockwork/releases) وفكّ ضغطه — بداخله ملف واحد هو `Clockwork.exe`؛ ضعه في أي مجلد (محمول — ضعه حيث شئت). لبنائه بنفسك، راجع **للمطوّرين** أدناه.
2. انقر نقرًا مزدوجًا على **`Clockwork.exe`** لفتح نافذة الإعدادات.
   - عند **التشغيل الأول** يحمّل بضعة **نماذج** في قائمة بدء التشغيل والتذكيرات لتكيّفها حسب احتياجك — وكلها غير محدَّدة في البداية، فلا يُنفَّذ شيء حتى تحدّده بنفسك. تُحفظ إعداداتك في `clockwork.settings.json` بجوار الملف التنفيذي — محليًا فقط، ولا تُرفع إلى المستودع.
3. لتشغيله عند كل إقلاع: في تبويب **الإعدادات**، انقر **البدء عند تسجيل الدخول** (يسجّل مهمة مجدولة بصلاحيات مسؤول، فلا يظهر سيل من نوافذ UAC عند الإقلاع).

> يبقى بهدوء في الشريط. انقر نقرًا مزدوجًا على أيقونة الشريط لفتح النافذة؛ زر الإغلاق يخفيها إلى الشريط فقط. للخروج فعليًا، انقر بالزر الأيمن على الشريط واختر **خروج**.

## التبويبات الخمسة

### قائمة بدء التشغيل

**قائمة خطوات مرتّبة** تُنفّذ من الأعلى إلى الأسفل عند تسجيل الدخول. انقر **إضافة ▾** لاختيار نوع؛ أضف/احذف/أعد الترتيب بحرية؛ يمكن تفعيل/تعطيل كل خطوة، وإعطاؤها **تأخيرًا بعد الخطوة**، و**عدد تكرارات** (كرّرها N مرة)، وشروطًا (**في أيام أسبوع محددة فقط / قبل الساعة N فقط**). أنواع الخطوات:

- **تشغيل برنامج** — الهدف (**استعراض…** لاختيار ملف) / الوسائط / مجلد العمل (اتركه فارغًا = مجلد الهدف) / مسؤول. يمكن أن يكون الهدف `.exe` أو مستندًا أو اختصارًا أو رابطًا؛ ويُشغَّل `.ps1` عبر PowerShell. متقدم: **نمط النافذة** (مصغّرة / مكبّرة / مخفية)، **التنشيط إن كان يعمل بالفعل** (إبرازه إلى الأمام بدل إعادة التشغيل؛ اسم العملية عبر **اختيار…**)، **مسارات بديلة** (مسار كامل واحد لكل سطر؛ يُستخدم أول مسار موجود — مفيد حين تختلف مسارات التثبيت بين الأجهزة).
- **إرسال مفاتيح** — مثل Win+D، Alt+K، Ctrl+Enter، F5 (**التقاط** لتسجيل اختصار بالضغط عليه).
- **إرسال نص** — اكتب نصًا في النافذة النشطة (أو في **عملية هدف** مُختارة عبر **اختيار…**).
- **مستوى الصوت** — كتم / إلغاء الكتم / ضبط المستوى.
- **إجراء على نافذة** — حسب اسم العملية (**اختيار…**، قابل للبحث): إغلاق / تصغير / تكبير / إبراز إلى الأمام / الإبراز ثم إرسال مفاتيح؛ للتطبيقات البطيئة يمكن **الانتظار حتى N ثانية لظهور النافذة**.
- **أمر نظام** — إظهار سطح المكتب / قفل / إطفاء الشاشة / إفراغ سلة المحذوفات / مسح الحافظة / فتح الإعدادات / إدارة المهام / لقطة شاشة / سكون / إسبات / تسجيل الخروج / إعادة التشغيل / إيقاف التشغيل (الثلاثة الأخيرة تطلب تأكيدًا أولًا).
- **تأخير** — مجرد الانتظار N ثانية قبل الخطوة التالية.
- **مجموعة إجراءات** — شغّل مجموعة إجراءات معرّفة؛ اضبط عدد التكرارات لتكرار المجموعة كاملة.

> **تأخير بدء التشغيل** (تبويب الإعدادات، عند الإقلاع فقط): انتظر عددًا ثابتًا من الثواني بعد تسجيل الدخول حتى تمرّ «عاصفة تسجيل الدخول» (تنافس القرص/المعالج من كل عناصر البدء التلقائي) قبل تشغيل القائمة؛ لا يتأثر التشغيل اليدوي. ارفعه (0–600 ثانية) إن بدأت الأمور مبكرًا جدًا.

> **الإيقاف في أي وقت** — الشريط ← **إيقاف الإجراءات الجارية**، أو **مفتاح الطوارئ** العام (يُضبط في تبويب الإعدادات؛ الافتراضي `Ctrl+Alt+Q`). يتوقف كل ما يعمل بعد الإجراء الحالي؛ وتُقاطَع الانتظارات الطويلة (تأخير بدء التشغيل، انتظار نافذة) فورًا.

### التذكيرات

اضبط **وقتًا** (أو بدّل إلى **عند تسجيل الدخول**)، و**تكرارًا** (أيام الأسبوع / كل N يوم / شهريًا)، و**النص**؛ واختياريًا اقرأه بصوت مسموع. التذكيرات التي لها إجراء **عند-نعم** (تشغيل برنامج / فتح ملف / رابط / تشغيل مجموعة إجراءات) تُظهر مربع **نعم / لا** مع زر **تأجيل** (الافتراضي 10 دقائق، قائمة ▾ من 5–60 دقيقة)؛ أما البقية فتنزلق في الزاوية على هيئة **بطاقة تذكير** (تُغلق تلقائيًا بعد الثواني المضبوطة، **0 = تبقى حتى تصرفها**). يمكنك أيضًا ضبط **مجموعة إجراءات صامتة** — تشغيل مجموعة في وقتها دون أي نافذة منبثقة.

متقدم: **الإغلاق التلقائي**، **الإلحاح المتكرر** (إعادة الظهور كل N دقيقة حتى موعد نهائي)، **تأخير بعد التشغيل + تشتيت عشوائي**، **مهلة سماح** (التقاط تشغيل فاته إيقاف/سكون قصير)، **التعويض عند الفوات** (إعادة التشغيل مرة واحدة بعد أن فوّته الإسبات/الإيقاف)، و**تاريخ مرجعي** لتكرار كل N يوم (**اختيار التاريخ**). تبقى حالتا «تم التشغيل اليوم» و«مؤجّل حتى» عبر عمليات إعادة التشغيل (`clockwork.state.json`)، فيستمر التأجيل عبر إعادة التشغيل ولا يتكرر أي تنبيه مرتين.

بحاجة إلى التركيز أو حضور اجتماع؟ يوفّر الشريط **إيقاف التذكيرات مؤقتًا لمدة 1 / 2 / 4 ساعات** (عدم الإزعاج): يُكبَت كل شيء (بما في ذلك المجموعات الصامتة) ويُستأنف تلقائيًا عند انتهاء الوقت.

### عناصر بدء تشغيل النظام

تعرض **كل ما يبدأ تلقائيًا** (مفاتيح Run في السجل، مجلدات Startup، المهام المجدولة). أزل تحديد **تمكين** لإيقاف عنصر — **مُعطّل، غير محذوف؛ أعد التحديد للاستعادة** (يسري فورًا). العناصر المعلّمة بـ **يتطلب مسؤولًا** تطلب إعادة الفتح بصلاحيات مرتفعة. عناصر النظام / السياسة / مرة-واحدة (Group-Policy Run، RunOnce، Winlogon، Active Setup) لا يمكن المساس بها وهي **مخفية افتراضيًا** — علّم **إظهار عناصر النظام / للقراءة فقط** لعرضها (باهتة). انقر بالزر الأيمن على صف لتجد **نقل إلى قائمة البدء** (يسلّم العنصر إلى Clockwork؛ مفاتيح Run في السجل وعناصر مجلد Startup فقط) أو **حذف من النظام** (يزيل المُدخل نهائيًا — يطلب تأكيدًا أولًا، ولا يمكن التراجع عنه؛ إلغاء التحديد هو الخيار القابل للتراجع). يبحث **مرشّح** في الأعلى بالاسم / الأمر؛ مرّر المؤشر فوق أمر مقتطع لقراءته كاملًا.

### مجموعات الإجراءات

اجمع الإجراءات في مجموعة قابلة لإعادة الاستخدام. يبدأ **إضافة ▾** مجموعة من **قالب جاهز** (تركيز / اجتماع / إنهاء اليوم / قبل النوم / مغادرة قصيرة / لقطة شاشة) — عدّل أسماء العمليات ثم احفظ. المجموعة **تعرّف الإجراءات فقط**؛ شغّلها بأربع طرق: من الشريط (**تشغيل: <المجموعة>**)، **مفتاح عام**، كـ **خطوة مجموعة إجراءات** في قائمة بدء التشغيل (عند الإقلاع)، أو من تذكير (**عند-نعم / مجموعة صامتة**). لا تشغّل المجموعة سوى نسخة واحدة في المرة؛ ويمكن أن تعمل خطوة **رسالة** كبوابة تأكيد (الإجابة بـ **لا** تلغي البقية).

> **مفتاح عام** — في محرّر المجموعة، انقر مربّع المفتاح واضغط اختصارًا (مثل `Ctrl+Alt+F`) لتشغيل تلك المجموعة من أي مكان دون الحاجة إلى قائمة. Esc للإلغاء، Delete للمسح. المجموعات المعطّلة تحرّر تركيبتها؛ التركيبات المحجوزة للنظام (Alt+F4، Ctrl+Shift+Esc…) والتركيبات المستخدَمة بالفعل من مجموعة أخرى أو من مفتاح الطوارئ تُرفَض مع إشعار.

### الإعدادات

**تأخير بدء التشغيل** (0–600 ثانية، عند الإقلاع فقط)، **البدء مصغّرًا في الشريط**، **مفتاح الطوارئ** (انقر المربع واضغط اختصارك؛ Esc للإلغاء، Delete للمسح؛ الافتراضي `Ctrl+Alt+Q`)، و**لغة الواجهة** (الصينية المبسّطة، English، 日本語 و15 أخرى — 18 إجمالًا؛ يعيد التبديل تشغيل التطبيق لتطبيقها).

**تصدير الإعدادات / استيراد الإعدادات** — انقل إعداداتك كاملة إلى حاسوب آخر أو احتفظ بنسخة احتياطية. التصدير يكتب نسخة من `clockwork.settings.json` في أي مكان تختاره؛ أما الاستيراد فيستبدل **كل شيء** (قائمة بدء التشغيل / التذكيرات / مجموعات الإجراءات / الإعدادات)، لذا يطلب تأكيدًا أولًا، ويحتفظ بنسخة احتياطية من الإعدادات الحالية في `clockwork.settings.json.bak`، ثم يعيد تشغيل التطبيق لتطبيقها.

## نصائح

- **انقر نقرًا مزدوجًا على صف لتحريره.** عند إدخال المسارات / العمليات / الاختصارات / التواريخ لست مضطرًا للكتابة يدويًا: **استعراض…**، **اختيار…** (منتقي عمليات قابل للبحث)، **التقاط**، و**اختيار التاريخ**.
- **تكرار** (تبويبا التذكيرات / مجموعات الإجراءات) يستنسخ الصف المحدَّد أسفله مباشرة — أسرع من إعادة بناء عنصر شبه مطابق؛ وتُسمّى المجموعة المكرَّرة «… (نسخة)».
- **الحذف يطلب تأكيدًا أولًا دائمًا** في كل مكان — صفوف القوائم، والخطوات داخل محرّر المجموعة، وعناصر بدء تشغيل النظام.
- النقر المزدوج على `Clockwork.exe` يفتح الإعدادات فقط — ولا يشغّل قائمة بدء التشغيل فورًا؛ استخدم **إعادة تشغيل قائمة بدء التشغيل** من الشريط لذلك.
- **شغّله بالطريقة العادية** (نقر مزدوج / الشريط / مهمة مجدولة). بعض المُشغّلات المعزولة / محدودة الصلاحيات تحجب النداءات منخفضة المستوى، فقد لا تعمل إرسال المفاتيح / إجراءات النوافذ / التنشيط إن كان يعمل / إرسال نص إلى عملية / مستوى الصوت (ستصلك رسالة واضحة؛ لا يتأثر «تشغيل برنامج» البسيط).
- إعداداتك في `clockwork.settings.json` (محليًا فقط). احذفه لإعادة التعيين إلى النموذج. حالة التذكيرات في `clockwork.state.json` (محلي أيضًا؛ حذفه آمن).
- تتطلب إضافة خطوة `.ahk` تثبيت AutoHotkey. المفاتيح المختصرة العامة / توسيع النص خارج النطاق — تلك قوة AutoHotkey.

## للمطوّرين

C#/.NET WPF؛ المصدر في `app/` (يتطلب .NET 10 SDK). الطبقات: `Core/` منطق صرف · `Native/` تفاعل Win32 · `Engine/` تنفيذ · `ViewModels/` + `Views/` الواجهة · `I18n/` + `Resources/` التوطين (المحايد = مصدر صيني، ملف `Strings.<code>.resx` تابع لكل لغة).

- تشغيل الاختبارات (xUnit):
  ```powershell
  dotnet test app.Tests/Clockwork.Tests.csproj
  ```
- بناء ملف exe واحد قائم بذاته (خصائص single-file / self-contained / الضغط مضبوطة في csproj):
  ```powershell
  dotnet publish app/Clockwork.csproj -c Release -r win-x64
  ```
  الناتج: `app/bin/Release/net10.0-windows/win-x64/publish/Clockwork.exe`.
- **CI / الإصدارات** (GitHub Actions): يبني push / PR ويشغّل كل الاختبارات على Windows runner؛ ودفع وسم `v*` (مثل `v2.0.0`) يبني، ويطبع إصدار الملف من الوسم، وينشئ GitHub Release ويرفق `Clockwork-<الوسم>.zip` (يحتوي على `Clockwork.exe`).
