# Clockwork 使用说明

[English](USAGE.md) · **中文** · [← 返回 README](../README.md)

电脑上重复的事，自动帮你做：开机自动开好软件 · 按时提醒 · 一键完成一串操作。

一个 Windows 托盘小工具，管好开机与日常四件事（外加一个「设置」页）：

1. **我的启动清单** —— 开机自动按顺序打开常用软件、顺手做几件杂事。
2. **定时提醒** —— 到点提醒 / 朗读 / 重复催促 / 点「是」后做事。
3. **系统启动项** —— 查看并管理电脑里所有开机自启项，一键关闭不需要的。
4. **动作组** —— 把一串动作打包成「组」（专注 / 收工 / 睡前……），一键触发，或绑**全局热键**随手唤起。

界面提供 **18 种语言**，首次运行默认跟随系统显示语言（可在「设置」页手动切换）。

---

## 怎么开始

1. 解压到任意文件夹（绿色免安装，放哪都行）。
2. 双击 **`Clockwork.exe`** 打开设置窗口。
3. 想每次开机自动运行：到 **「设置」** 页点「开机自启」按钮开启（用管理员权限注册计划任务，开机时不会弹一堆授权框）。

平时它安静待在右下角托盘。点窗口的关闭按钮只是收起到托盘，不会退出；彻底退出用托盘右键 **「退出」**。在「设置」页勾选 **「启动时最小化到托盘」** 后，手动打开也直接进托盘、不弹主窗口。

## 第一次用：把示例换成你自己的

首次打开，清单 / 提醒 / 动作组里已有一批**示例**（都带「示例」字样），只是演示玩法、给你照葫芦画瓢用的——放心大胆改或删。启动清单里前几条默认勾选、开机会执行；末尾的「Windows 设置 / 任务管理器」等默认没勾选，只当样例摆着，勾上才执行。

最常见的「开机自动打开我常用的几个软件」，这样弄：

1. 切到 **我的启动清单**。
2. 把用不上的示例整条删掉（选中 → 删除，删完自动选中下一条，连着删很顺手）。
3. **「新增 ▾」→ 启动程序**，「目标」填你要开的软件：
   - 系统能找到的填程序名即可：`msedge.exe`（Edge）、`notepad.exe`（记事本）；
   - 找不到的填**完整路径**：右键该程序 →「打开文件所在位置」→ 右键图标「属性」，把里面的「目标」路径复制粘进来；
   - 网址（`https://…`）、文档、`.ps1` 脚本、快捷方式（`.lnk`）也都能填。
4. 想某个软件晚点再开，就把它的 **「执行后延时」** 调大几秒，或上下移调顺序。
5. 托盘图标右键 → **「重新运行启动清单」** 先试跑一遍，确认都对。
6. 满意后到 **「设置」** 页开启「开机自启」，以后开机自动照做。

> 只想用「定时提醒」或「动作组」？去对应标签页照同样思路改示例即可，启动清单可全删空——四块功能各自独立，用哪块开哪块。

## 我的启动清单

- 一条**有序步骤表**，开机从上到下依次执行。可增删、上移 / 下移；**双击条目即编辑**。
- 每步可单独启停、设「执行后延时」、设「重复次数」（循环动作：连续执行 N 次，每次之间等「执行后延时」），并设条件「仅某些星期 / 仅当天 N 点前」。
- 选中一条按 **「运行」** = 立即只跑这一步看效果（无视该项启停与时间条件，纯测试）；结果弹托盘气泡反馈。

### 步骤类型

| 类型 | 说明 |
| --- | --- |
| **启动程序** | `.exe` / 文档 / 快捷方式 / 网址都行（目标可「浏览…」选文件）；`.ps1` 自动用 PowerShell 运行。工作目录留空 = 目标所在目录。进阶：**窗口风格**（最小化 / 最大化 / 隐藏启动）、**已在运行则激活**（把它带到最前、不重复启动；进程名可「选择…」）、**备用路径**（每行一条完整路径，主路径不存在时用第一个存在的——多设备安装路径不一致时用）。 |
| **发送按键** | 如 Win+D、Alt+K、Ctrl+Enter、F5（支持 Enter / Tab / Esc / Del / 方向键等键名；可点「捕获」按快捷键自动填）。 |
| **发送文本** | 往当前焦点窗口逐字输入一段文本（换行 = 回车、Tab 生效）；可选「目标进程」（可「选择…」）——填了先把它的窗口带到最前再输入，留空则发给当前焦点窗口。 |
| **音量** | 静音 / 取消静音 / 设为音量（设音量会先自动取消静音）。 |
| **窗口动作** | 按进程名（可「选择…」，带搜索）关闭 / 最小化 / 最大化 / 带到最前面 / 置前并发送按键；慢启动应用可设「等窗口出现最多 N 秒」，窗口一出现才动手、不盲等固定延时。 |
| **系统命令** | 显示桌面 / 锁屏（回来需输密码）/ 息屏（只关屏幕，动鼠标即亮）/ 清空回收站 / 清空剪贴板 / 打开 Windows 设置 / 打开任务管理器 / 截图 / 睡眠（秒醒）/ 休眠（断电保存）/ 注销 / 重启 / 关机（后三项执行前需确认）。 |
| **延时** | 单纯等待 N 秒再继续下一步；放清单最前面可整体推迟启动。 |
| **动作组** | 引用一个已定义好的动作组；设「重复次数」即可把整组动作循环 N 遍。 |

### 开机延迟

在「设置」页的「开机延迟 N 秒」框，**仅开机自启时生效**。登录后先固定等这么多秒，让「登录风暴」（磁盘 / CPU 被一堆自启抢占）过峰，再跑清单；手动「重新运行」不受影响。开机太早、有软件没起来就把它调大（0–600 秒）。这是整体延时的**唯一旋钮**；只想让某一步慢一点则用那步的「执行后延时」。

### 随时叫停

托盘右键 → **「停止正在运行的动作」**，或按全局**急停键**（在「设置」页设置；默认 `Ctrl+Alt+Q`）。正在跑的启动清单 / 动作组 / 单步会在当前动作做完后立刻停下（循环设多了不用干等）；开机延迟、等窗口出现等长等待也会被立即打断。日志如实记录「已手动停止」。热键被别的程序占用而注册失败时会弹托盘气泡提醒（此时用托盘菜单的「停止」兜底）。

> **进阶：** 想「等网络 / 桌面就绪再跑」而非固定等待，可在 `clockwork.settings.json` 把 `startupWaitForReady` 设为 `true`（默认 `false`，就绪即走、封顶 90 秒）。

## 定时提醒

- **触发：** 定时（到点）或 **登录时**（可设「仅开机 N 分钟内」才算登录——新建默认 10 分钟）。
- **周期：** 按星期 / 每 N 天 / 每月某号；提醒可朗读。
- 没配动作的提醒 → 走右下角**提醒卡片**（不置顶抢视线）；显示多久由「自动关闭」秒数决定，**0 = 不自动关、常驻到你点掉为止**——离屏一会儿也不会错过。重复催促型仍用弹窗，好一键停催。
- 配了「点是后」动作（运行程序 / 打开文件 / 开网页 / 跑动作组）→ 弹置顶 **「是 / 否」** 窗，还有「稍后」可打个盹（默认 10 分钟，旁边 ▾ 可选 5 / 10 / 15 / 30 / 60 分钟）。
- **进阶：** 自动关闭 · 重复催促（没确认就每 N 分钟再弹、直到某时刻）· 到点后延迟 + 随机延迟 · 宽限（到点时恰好关机 / 睡眠，N 分钟内补弹）· **错过必补**（休眠 / 关机把提醒整段错过时，下次程序在跑就补弹一次）· 「每 N 天」的**起算日**（可「选日期」）。
- **状态落盘：** 「当天已提醒过」「稍后到几点」记在 `clockwork.state.json`，重启程序也不丢——打的盹跨过重启仍会响，同一条提醒一天也不会重复弹。
- **勿扰：** 托盘右键 **「暂停提醒 1 / 2 / 4 小时」**，期间所有提醒（含静默动作组）都不触发，到时自动恢复，也可随时点「恢复提醒」；错过的按宽限 / 错过必补的正常规则处理。
- **静默动作组：** 到点不弹窗、直接跑指定动作组。选中提醒按「运行」立即试跑一次——注意：静默提醒的「运行」会**真的执行**那个动作组。

## 系统启动项

- 列出电脑里**所有开机自启**项（注册表 Run 键 / 启动文件夹 / 计划任务）。
- 取消「启用」勾选即关闭——**只是禁用、不删除，随时勾回来恢复**（即时生效）。
- 带「需管理员」的项：操作时会提示以管理员身份重新打开，同意后再操作即可。
- 系统 / 策略 / 一次性项（组策略 Run、RunOnce、Winlogon、Active Setup）无法常规开关、**默认不显示**——勾右上角「显示系统 / 只读项」才列出（置灰、仅供查看）。
- **「纳入启动清单」** 把某项交给本工具接管（自动停用原项 + 加进你的清单）。仅支持注册表 Run 键与启动文件夹项；计划任务暂不支持，会给出提示。
- 顶部**过滤框**可按名称 / 命令快速筛；命令行太长被截断时鼠标悬停可看全文。

## 动作组

- **「新增 ▾」** 可从**内置模板**（专注 / 会议 / 收工 / 睡前 / 离开一下 / 截图标注）创建，改成自己的软件再保存。
- 同一组同时只会跑一份，重复触发自动跳过。
- 四种触发：托盘 **「运行：某组」** · **全局热键** · 启动清单里加「动作组」步骤（开机跑）· 提醒挂「点是后 / 静默动作组」（定时跑）。
- **全局热键：** 在组编辑器里点热键框、按下组合（如 `Ctrl+Alt+F`），之后不开菜单、在任何程序里按键即跑该组。Esc 取消、Delete 清空；改动即时生效、无需重启。**禁用**的组会让出组合给别的组用。以下会被拒并提示：**系统保留**组合（Alt+F4、Alt+Tab、Ctrl+Shift+Esc……）、已被**其它启用组**或**急停键**占用、以及**被别的程序占用**（换一个组合即可）。
- 组里的「消息」步骤可当**确认闸门**：答「否」中止整组（比如收工前问「记录好了吗？」）。

## 设置

- **开机延迟**（0–600 秒，仅开机自启时生效）。
- **启动时最小化到托盘**（手动打开也直接进托盘、不弹主窗口）。
- **急停键**：点击输入框后按下想要的快捷键即录入，Esc 取消、Delete 清空停用；默认 `Ctrl+Alt+Q`。
- **界面语言**：简体中文、English、日本語 等共 18 种，切换后自动重启生效。

## 小提示

- 双击 `Clockwork.exe` 只打开设置窗口，**不会**立刻执行启动清单；要立刻跑用托盘「重新运行启动清单」。
- 个人设置存在文件夹内的 `clockwork.settings.json`（只在你自己电脑上）；删掉它重开程序即可恢复成示例配置。提醒状态存在 `clockwork.state.json`（同样只在本机，删掉无妨——最多当天的提醒再弹一次）。
- 填路径 / 进程 / 组合键 / 日期不用硬记：**「浏览…」「选择…」（带搜索）「捕获」「选日期」**；进程选择框和系统启动项列表都带搜索 / 过滤。
- 请用**正常方式启动**（双击 `Clockwork.exe` / 托盘 / 计划任务）。若用某些沙箱 / 受限权限的启动器（如 Lucy）打开，Windows 会限制部分底层调用，发送按键 / 窗口动作 / 已运行则激活 / 发送文本到指定进程 / 音量 等原生功能可能不可用（会给清楚提示，不影响单纯「启动程序」）。
- 全局热键可用来**触发动作组**（每组单独设，见上）。任意按键重映射 / 文本展开仍不在本工具范围（那是 AutoHotkey 的强项；加 `.ahk` 步骤需装 AutoHotkey）。
